# How to Set M1, M2, M3 Memory Presets on Sony A7C from P to Manual Mode

## Overview

The Sony A7C camera features three customizable memory preset modes (M1, M2, M3) that allow you to save and quickly recall your preferred camera settings. This guide will walk you through the process of configuring Manual (M) mode settings and saving them to these memory presets.

## Prerequisites

- Sony A7C camera
- Understanding of basic camera settings (aperture, shutter speed, ISO)

## Step-by-Step Instructions

### Part 1: Configure Your Manual Mode Settings

1. **Switch to Manual Mode**
   - Turn the mode dial on top of the camera to **M (Manual)** position
   - This gives you full control over aperture, shutter speed, and ISO

2. **Set Your Desired Settings**
   Configure the following settings according to your needs:
   
   - **Aperture (f-stop)**: Use the rear control dial to adjust
   - **Shutter Speed**: Use the front control dial to adjust
   - **ISO**: Press the ISO button and use the control dial to set
   - **White Balance**: Menu → Camera Settings → White Balance
   - **Focus Mode**: Use the focus mode switch or menu
   - **Drive Mode**: Press the drive mode button to select (Single, Continuous, etc.)
   - **Metering Mode**: Menu → Camera Settings → Metering Mode
   - **Image Quality/Size**: Menu → Image Quality Settings
   - **Picture Profile/Creative Style**: Menu → Camera Settings
   
3. **Fine-tune Additional Settings**
   - Set any other custom functions you use frequently
   - Configure custom button assignments if needed
   - Set up any specific autofocus settings

### Part 2: Save Settings to Memory Preset (M1, M2, or M3)

1. **Access the Memory Menu**
   - Press the **MENU** button
   - Navigate to: **Camera Settings** (camera icon) → **Memory**
   
2. **Register Your Settings**
   - Select **Memory** in the menu
   - Choose which preset to save to: **M1**, **M2**, or **M3**
   - Select **Register**
   - Confirm by selecting **OK**

3. **Verify the Save**
   - Your current settings are now saved to the selected memory preset
   - A confirmation message should appear

### Part 3: Using Your Memory Presets

1. **Recall a Preset**
   - Turn the mode dial to **M1**, **M2**, or **M3**
   - The camera will immediately load all saved settings

2. **Switch Between Presets**
   - Simply rotate the mode dial to the desired memory preset
   - All settings change instantly

## Recommended Memory Preset Configurations

Here are some popular preset configurations:

### M1: Low-Light Photography
- Mode: Manual
- Aperture: f/1.8 - f/2.8 (wide open)
- Shutter Speed: 1/60s or slower
- ISO: Auto ISO with high limit (3200-6400)
- Focus Mode: AF-S with Eye AF enabled

### M2: Daylight/Portrait
- Mode: Manual
- Aperture: f/2.8 - f/5.6
- Shutter Speed: 1/250s
- ISO: 100-400
- Focus Mode: AF-C with tracking

### M3: Action/Sports
- Mode: Manual
- Aperture: f/2.8 - f/4
- Shutter Speed: 1/500s or faster
- ISO: Auto ISO
- Drive Mode: Continuous High
- Focus Mode: AF-C with wide tracking area

## Tips and Best Practices

1. **Name Your Presets**: While the A7C doesn't allow custom naming, keep a mental or written note of what each preset is for

2. **Regular Updates**: Update your presets as your shooting style evolves

3. **Test Before Important Shoots**: Always verify your saved settings before critical photography sessions

4. **Backup Settings**: Consider documenting your preset configurations in case you need to reconfigure

5. **Mode Dial Position**: Remember that each preset (M1, M2, M3) is a physical position on the mode dial

## Troubleshooting

**Settings don't save**
- Ensure the camera is in Manual mode when saving
- Check that the battery has sufficient charge
- Try resetting the camera and reconfiguring

**Preset recalls unexpected settings**
- Verify which preset you saved to (M1, M2, or M3)
- Re-save the preset with your desired configuration

**Cannot access Memory menu**
- Ensure you're in a mode that supports memory registration
- Check that the camera firmware is up to date
